What's this?

In this view you can create Sprints and assign issues to your defined sprints. This will help you to structure your working process. All new issues will appear in the "Backlog" section of this page. When you click on an issue, you can add a label which represents the guessed amount of time you think the issue will take to solve. You can assign issues to milestones as well by using the input "typeahead".
